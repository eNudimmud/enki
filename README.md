## 🚀 Installation

### Prérequis
- Node.js v22.12+
- OpenClaw 2026.3+

### Setup

1. **Installer OpenClaw:**
   ```bash
   npm install -g openclaw@latest
   ```

2. **Cloner ce repo:**
   ```bash
   git clone https://github.com/eNudimmud/enki-openclaw.git ~/.openclaw/workspace
   cd ~/.openclaw/workspace
   ```

3. **Configurer:**
   - Éditer `USER.md` avec ton contexte
   - Mettre à jour `TOOLS.md` (environnement local)
   - Ajuster `SOUL.md` et `IDENTITY.md` si nécessaire

4. **Lancer:**
   ```bash
   openclaw gateway start
   ```

5. **Valider config:**
   ```bash
   openclaw config validate
   openclaw doctor
   ```
